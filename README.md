# Scoop Bucket — Zeri

Scoop manifests for [Zeri](https://github.com/ilmartotch/ZeriReplEngine), a TUI multi-language REPL.

## Install

```powershell
scoop bucket add zeri https://github.com/ilmartotch/scoop-zeri
scoop install zeri
```

## Upgrade

```powershell
scoop update zeri
```

## Uninstall

```powershell
scoop uninstall zeri
scoop bucket rm zeri
```
